Oracle Intelligent Bots Advanced Training - Workarounds and Troubleshooting

This page collects workarounds and troubleshooting information.

Can't Install ngrok on Windows

Depending on how your machine is configured, you might not be able to install ngrok. For example, Oracle's standard OBI image seems to block it for security reasons.

You can work around this using the Node localtunnel package. Like ngrok, the localtunnel package allows you to create a https:// URL for a local URL. You don’t need to subscribe anywhere. Each time you run the lt command, it creates a new host name.

Prerequisites for Setup

  • Have Node.js already installed.
  • Be outside of the Oracle network when installing (to avoid proxy issues).

Installation and Verification

  1. At the command line, enter: 
    npm install -g express-generator
  2. On your system, create a folder called MySimpleTest.
  3. Open a command prompt and navigate to MySimpleTest.
  4. Enter: 
    express
  5. Enter: 
    npm install
  6. Enter: 
    npm start
  7. In a web browser, enter the following: 
    localhost:3000

    You should see the message "Welcome to Express".

  8. Install localtunnel as explained here at https://www.npmjs.com/package/localtunnel.
  9. In the command prompt, enter: 
    lt --port 3000

    It should return a URL.

  10. Go back to the web browser and enter the URL that was just returned.

    You should get the same welcome message that you did before.

Important: Be sure to close the tunnel after finishing the lab. You do so by pressing Ctrl-c.

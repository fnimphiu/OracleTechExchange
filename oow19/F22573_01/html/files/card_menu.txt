This is a partial view of the Edit Message dialog for Bot Messages. In this image, the Cards option has been selected. There are two tabs, General and Cards (selected here). Within the Cards tab, there is a set of tiles for each card that has been defined for this message. There are three, each having a context menu. The first tile also bears a check mark, meaning that this card will display be default when you replay the conversation in the preview. In this image, the context menu for the second card has been selected.

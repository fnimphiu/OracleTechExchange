Advanced Training 2018

Oracle Digital Assistant is the successor and enhancement of Oracle Intelligent Bots. The advanced training material has been developed by product management to meet the internal and external needs for in-depth training that goes beyond mere functional and architectural description. The training material is still valid and applies to skill bots in Oracle Digital Assistant.

The hands-on resources include starter bots that can be imported into Oracle Digital Assistant as skill bots. To implement the training environment, you need access to Oracle Digital Assistant, the Oracle Mobile Cloud (OMCe, AMCe or MCS), and the Oracle Application Container Cloud Service. The exact installation steps are described in a README document, which is located in the ZIP file that you can download from a link below.
